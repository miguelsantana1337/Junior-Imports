# Junior Imports — E-commerce V3

Projeto demonstrativo de e-commerce com vitrine, carrinho, checkout simulado e painel administrativo completo.

## Como abrir

1. Extraia o arquivo ZIP.
2. Abra `index.html` no navegador.
3. No rodapé, clique em **Abrir painel administrativo**.
4. Use o PIN padrão: `1234`.

## Recursos da loja

- banners rotativos;
- catálogo com busca, filtros e ordenação;
- produtos em destaque;
- preços, promoções, estoque e avaliações simuladas;
- favoritos;
- carrinho com quantidade e cupom;
- frete demonstrativo;
- pagamento simulado por Pix, cartão ou boleto;
- criação de pedido com código próprio;
- confirmação e envio do código pelo WhatsApp.

## Recursos do painel

- dashboard com pedidos, receita e ticket médio simulados;
- adicionar, editar, ocultar e excluir produtos;
- upload local ou URL de imagens;
- controle de preço, preço anterior, estoque, etiqueta e destaque;
- reordenação de produtos por botões ou arrastar e soltar;
- adicionar, editar, ocultar, excluir e reorganizar banners;
- gerenciamento de categorias;
- ordem e visibilidade das seções da página inicial;
- gerenciamento de cupons;
- consulta e alteração do status dos pedidos;
- configurações de WhatsApp, frete, Pix, cor, anúncio e PIN;
- exportação e importação dos dados em JSON;
- restauração dos dados padrão.

## Dados

Todos os dados são armazenados no `localStorage` do navegador. Portanto:

- alterações ficam apenas no navegador utilizado;
- limpar os dados do navegador apaga as configurações;
- imagens enviadas pelo painel são armazenadas em base64;
- use **Dados e backup → Exportar JSON** para salvar uma cópia.

## Aviso

Este é um projeto acadêmico e demonstrativo. Nenhuma venda, cobrança, pagamento ou entrega é processada de forma real.
